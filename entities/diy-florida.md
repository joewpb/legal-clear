# DIY Florida

Free interactive court form builder built into the Florida Courts E-Filing Portal.
Created by the Florida Courts E-Filing Authority (represents all 67 county clerks).

## How It Works

1. Register at [myflcourtaccess.com](https://www.myflcourtaccess.com) as "Self-Represented Litigant"
2. Click the **DIY** tab
3. Select your case type
4. Answer interview-style questions
5. Forms are auto-filled and ready to e-file

## Available Interviews (5 areas)

| Area | Description |
|------|-------------|
| Landlord Tenant Cases | Evictions, tenant disputes |
| Small Claims | Claims up to $8,000 |
| Domestic Violence | Protection injunctions |
| Simplified Dissolution | Simple divorce (no kids, no disputes) |
| Domestic Relations | Family law matters |

More areas are added monthly. Currently 20+ forms available.

## Key Advantage

DIY Florida auto-fills forms AND can e-file them directly. A pro se litigant
never needs to download a PDF, fill it by hand, and figure out where to file.

## Limitations

- Only covers 5 case types (expanding)
- Does not cover: probate, guardianship, complex divorce, name change, expungement
- Requires internet access and basic computer literacy
